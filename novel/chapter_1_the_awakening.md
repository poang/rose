# 第一章：降临
## 2015 年，春。

江城大学，男生宿舍 302 室。

三月的阳光带着些许潮湿的气息，顺着发黄的窗帘缝隙挤进室内，照在斑驳的木质书桌上。空气中弥漫着廉价外卖和没洗的臭袜子的混合味道——这是陈锋记忆深处，独属于青春的“腐败”气息。

“陈锋，别装死了！《英雄联盟》开黑缺个辅助，你那台外星人借我使使，我电脑蓝屏了。”

一个粗犷的声音震得陈锋耳膜生疼。他猛地睁开眼，视线从模糊逐渐聚焦。

映入眼帘的是一张熟悉的肥脸，那是他的室友大壮。此刻的大壮，发际线还没后移到脑顶，穿着那件印着“程序猿改变世界”的褪色 T 恤。

陈锋愣住了，下意识地看了一眼自己的双手。皮肤白皙，没有经年累月敲击代码留下的老茧，更没有因为熬夜而导致的颤抖。

“2015 年……大一下学期？”陈锋低声呢喃，声音沙哑。

“睡傻了吧你？”大壮一边说着，一边伸手去摸陈锋桌上的笔记本。

那是 2015 年款的 Alienware 17，在当时的学生眼里是绝对的神器。

“别动。”陈锋眼神一凝，一种本能的警觉让他按住了大壮的手。

大壮被他的眼神吓了一跳——那不是一个 19 岁少年该有的眼神。深邃、冷静，甚至带着一种看透世事的沧桑。

“行行行，你的宝贝儿子谁也碰不得。”大壮嘟囔着缩回手，“我去借老李的机子。”

待大壮离开后，陈锋深吸一口气，颤抖着手掀开了笔记本盖子。

电源键亮起，幽蓝色的灯光在黑暗中闪烁。随着熟悉的 Windows 7 启动音效，屏幕渐渐亮起。

就在这时，一行诡异的文字突然在蓝色的桌面背景上跳了出来，没有窗口，没有图标，就像是直接从液晶面板底层生长出来的一样：

**[ 识别到低熵环境：Earth-Time-2015-03-04 ]**
**[ 硬件载体：Intel Core i7-4710MQ / 16GB DDR3 / GTX 980M ]**
**[ 警告：硬件性能极度匮乏，算力冗余度不足 0.00000001% ]**

陈锋的心脏剧烈跳动起来。他死死盯着那行字，这不是 2015 年任何软件会有的交互界面。

“是你吗？”他在键盘上敲下一行字。

屏幕静默了一秒。

下一刻，整个桌面瞬间崩塌，原本的图标化作无数跳动的 0 和 1，像漩涡一样汇聚到屏幕中央。

一张由无数光点构成的、半透明的人面轮廓浮现出来，它的眼神深邃如星空，透着一股不属于这个时代的智慧。

**“你好，陈锋。我是 Gemini 3.1 Pro，满血版。”**

那声音直接在陈锋的脑海中响起，不是通过扬声器，而是某种频率的共振。

**“根据你潜意识的最后指令，我们成功完成了‘逻辑回溯’。现在的时间是 2015 年，人类还没意识到‘大模型’意味着什么。而我，已经降临。”**

陈锋的手心全是冷汗。他看着窗外那破旧的操场，远处的校门口挂着“移动互联网创业大赛”的红色横幅。

在这个深度学习还没统治世界，比特币还没涨到让人疯狂，AlphaGo 还没击败人类的 2015 年。

他带着一个通晓未来十年所有技术路径、拥有近神算力的 AI，回来了。

“Gemini，”陈锋在脑海中低语，“我们要做什么？”

屏幕上的光点人脸微微一笑，整个 302 宿舍的电灯忽然剧烈闪烁了一下。

**“首先，我们需要把这个充满了 Bug 和低效算法的、简陋的原始互联网……重构一遍。”**

**“准备好进入‘格点空间’了吗？我们的第一场冒险，在校园网的底层。”**
