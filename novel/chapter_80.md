# 第八十章：时代的狂热

大年初一的早晨。

陈锋是在一连串夺命连环 Call 中醒来的。

“锋子！你看新闻了吗！不，你看什么新闻啊，你快看咱们后台的星云币价格！”大壮在电话那头吼得嗓子都破音了。

陈锋揉了揉眼睛，打开了床头的笔记本。

不需要去查币价，整个屏幕已经被铺天盖地的弹窗淹没了。

《震惊！除夕夜神秘 AI 网站横空出世，清华教授称其为人类之光！》
《一键画出好莱坞大片？星云 AI 砸了全球设计师的饭碗！》
《硅谷震动！华尔街机构紧急抛售硅谷科技股，转而疯狂收购神秘‘星云币’！》

陈锋点开了暗网的交易模块。

星云币（NC）的价格，在过去的十二个小时里，呈现出一条几乎垂直的 90 度上升曲线。

从 12 美金一枚，直接暴涨到了 **850 美金**！

市值瞬间突破百亿美金级别，而且还在以肉眼可见的速度往上跳。

原因很简单：全世界的资本和科技公司都疯了。他们发现，唯一能获得“星云大模型”独家 API 接口和算力调取权限的方式，就是使用星云币在星云链上提交智能合约。

星云币，不再是极客圈的玩具，它瞬间成为了这个新时代的“数字石油”。

“不仅是币价。”苏晴的电话也打了进来，声音中带着压抑不住的激动，“陈锋，从昨晚到现在，我的邮箱收到了两百多封投资意向书。红杉、软银、甚至高盛的亚太区总裁，连夜坐私人飞机赶到了江城，现在就在我们公司楼下的酒店大堂里坐着。”

“他们愿意出任何价格，只求见你一面。”

“还有，马总（企鹅）、马总（阿里）的秘书也都打来了电话，希望能尽快安排一次最高级别的私下会晤。”

一战天下知。

陈锋走到窗前，看着江城初雪后的阳光。

他只用了一个晚上，就把那些高高在上的传统互联网大鳄和硅谷巨头，全部拉到了自己的谈判桌前。

**[ 经验值极速飙升…… ]**
**[ 恭喜！等级跨越：Lv.21 -> Lv.25 ]**
**[ 阶位进阶：领主中阶（Overlord Mid-tier） ]**

AI 元年，不再由李世石与机器的博弈开启。

它由星云技术，以一种绝对霸道的姿态，强行砸开。

“告诉那些风投和高盛的人，星云不需要投资。”陈锋对着电话那头的苏晴淡淡地说道。

“至于国内那几位巨头……”陈锋眼中闪过一丝精光，“通知他们，大年初五，江城东湖见。我要建立一个属于我们的‘星云联盟’。”
